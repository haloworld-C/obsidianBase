### glog
#### basic concept
glog是基于宏的, 可以很方便的使用串流输出, 并且自动添加换行符号，完全可以替代`std::cout`
log的级别: INFO, WARNING, ERROR
#### 基本使用
```cpp
#include <glog/logging.h>

LOG(INFO) << "something";
LOG(WARNING) << "something";
LOG(ERROR) << "something";
LOG_EVERY_N(INFO, 10) << "something"; //适用与重复内容

```
